# Advertising Data ETL-V

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Server

### How to install and run

1. `npm install`  
2. `npm run start:api`

### Defaults

`PORT=3000`

## Client

### How to install and run

1. `cd frontend`
1. `npm install`
2. `npm start`

### Defaults

`PORT=4000`
